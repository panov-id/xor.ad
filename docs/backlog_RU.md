# Бэклог идей (на будущее)

Короткие заметки на потом — не в текущей работе, чтобы не потерять.

## Заморозка сессии (self-lockout)

В приложении: кнопка «заморозить» — пользователь сам запрещает себе входить на какое-то время (напр. 20 минут), чтобы не залипать. На время заморозки вход не пускает.

Наброски:
- Пользователь выбирает длительность (пресеты: 20 мин / час / …).
- До истечения таймера приложение не даёт войти в ленту/чат — показывает экран «заморожено, осталось MM:SS».
- **Против обхода**: срок хранить на сервере (привязка к UID-браузеру), а не только в localStorage — иначе чистка кеша снимает блок. Учесть, что приватное окно = новая личность (см. модель регистрации), так что это «мягкий» самоконтроль, не жёсткий барьер.
- Настраиваемые пресеты/максимум — через env, как остальные параметры.

Открытые вопросы: можно ли снять заморозку досрочно; заморозка на витрину или на весь xor; поведение при истечении (сразу пускать / уведомление).

## Совместные игры без правил

В чате/мэтче — общая визуальная доска для двоих: **домино, шашки, шахматы**. Фишка: **никаких зашитых правил** — движок только рисует доску и даёт двигать/ставить фигуры свободно; правила игроки придумывают и соблюдают сами. Это ice-breaker и «общий момент», а не соревнование.

Наброски:
- Общая доска синхронизируется в реальном времени (Supabase Realtime, вебсокеты — через api-прокси).
- Набор: домино, шашки/шахматы (одинаковая механика «свободно двигай фигуры»), легко добавить новые доски.
- Никакой валидации ходов, счёта, победителя — только состояние доски + перетаскивание.
- Живёт в рамках чата/мэтча; исчезает вместе с ним (эфемерность).
- **Запуск/переключение — по запросу**: в чате кнопка «предложить игру» → выбор доски (домино/шашки/шахматы) → другому уходит запрос → принял → доска открывается у обоих. Смена игры — тем же запросом. Раз правил нет, согласие на переход — единственная «механика».

Открытые вопросы: сброс/перестановка доски; кто может двигать чужие фигуры (оба — да, раз правил нет); наборы фигур/тем под бренд; нужен ли простой «бросок кубика»/перемешивание для домино.
